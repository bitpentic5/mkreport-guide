# mkreport-guide

本仓库用于归档和发布多个独立 HTML 页面。仓库本身不绑定任何特定域名或网站，仅作为静态内容集合的托管与分发入口。

## 项目简介

mkreport-guide 是一个轻量级 HTML 页面归档仓库。所有页面均为独立 HTML 文件，不依赖外部数据库或后端服务。仓库结构清晰，便于快速浏览、检索与引用。

## 目录说明

```
mkreport-guide/
├── index.html                # 仓库首页，提供导航与索引
├── pages/                    # 存放所有 HTML 页面文件
│   ├── example-page.html     # 示例页面（可根据实际内容替换）
│   └── ...
└── assets/                   # 公共资源文件（可选）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的入口页面，建议在此列出所有归档页面的链接与简要说明。
- `pages/`：所有独立 HTML 页面均放置在此目录下，每个文件代表一个完整页面。
- `assets/`：存放页面共用的样式、脚本或图片资源（非必需）。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文档，可单独通过浏览器打开使用。
- 所有页面内容保持原始状态，不进行动态修改或重定向。
- 归档页面可能包含内部链接，但不会指向本仓库之外的外部资源（除非页面本身需要）。

## 维护说明

- 本仓库由维护者定期更新，新增或调整页面内容。
- 提交内容时请遵循以下原则：
  - 页面文件命名使用小写英文与连字符，例如 `report-2024.html`。
  - 避免在页面中包含任何形式的推广、营销或诱导性内容。
  - 确保页面在不同浏览器中基本可用。
- 欢迎通过 Issue 或 Pull Request 提出改进建议，但本仓库不接受外部直接提交的页面内容。
